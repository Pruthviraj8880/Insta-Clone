# Instagram Clone - React Project

A fully responsive Instagram clone built with React.js for the WAP Group Project.

## 🚀 Features
- Responsive Design: Works on Mobile, Tablet, and Desktop.
- Story Bar: Gradient-ringed story circles with horizontal scroll.
- Interactivity: Double-tap to like and functional "Like" toggle.
- Comment System: Real-time comment posting and auto-focusing input.
- Suggestions Sidebar: Desktop-only sidebar for user discovery.

## 🛠️ Tech Stack
- Library: React (JSX)
- Icons: Lucide-React
- Styling: Custom CSS3 with Flexbox & Media Queries
- State Management: React Hooks (`useState`, `useRef`, `useEffect`)

## 📦 Installation
1. Clone the repo: `git clone [your-link]`
2. Install dependencies: `npm install`
3. Start the app: `npm start`
